<b>יוצר .</b> בורא : <b>גובי .</b> ארבה כי ויעל הארבה ( שמות ט' ) ת''א גובא , וכן כגוב גובי ( נחום ג' ) : <b>הלקש .</b> כן יקרא העשב המאוחר כמו שנקרא מטר המאוחר בשם מלקוש : <b>גזי .</b> מל' גזיזה וקצירה :
! {ג}
<b>נחם .</b> ענין הפוך מחשבה :
! {ד}
<b>תהום .</b> עומק המים : <b>רבה .</b> גדולה בעומק : <b>החלק .</b> כן יקרא השדה וכן חלקה אחת תמטר ( לעיל ד' ) :
! {ה}
<b>חדל .</b> ענין מניעה :
! {ז}
<b>אנך .</b> כן יקרא קו הבנין ובו משקולת הבדיל וע''י מיישר הבנאי בנין החומה ואנך הוא בדיל בל' ערבי :
! {ח}
<b>עבור .</b> מל' העברה :
! {ט}
<b>ונשמו .</b> מל' שממון : <b>במות .</b> מלשון במה ומזבח : <b>ישחק .</b> כמו יצחק ויתכן שאמר ישחק לרמז שהבמות יהיו לשחוק וללעג : <b>ומקדשי .</b> ענין ארמון משגב וכן למטה מקדש מלך הוא וכמו והיה למקדש ( ישעיה ח' ) : <b>יחרבו .</b> מל' חורבן :
! {י}
<b>כהן .</b> ר''ל כהן הבעל וכומר לעכו''ם : <b>קשר .</b> ענין אגודת מרד כמו ויהי הקשר אמיץ ( ש''ב טז ) : <b>להכיל .</b> לסבול כמו נלאיתי הכיל ( ירמי' ו' ) :
! {יב}
<b>חוזה .</b> רואה ונביא :
! {יג}
<b>מקדש .</b> ארמון משגב :
! {יד}
<b>בוקר .</b> מל' בקר : <b>ובולס .</b> ענין בלילה וערבוב ובדרז''ל שכן עני אוכל פתו מעיסה בלוסה ( שבת עו ) : <b>שקמים .</b> שם מין ממיני התאנים וכן ושקמותם בחנמל ( תהלים ע''ח ) :
! {טז}
<b>תטיף .</b> הוא מל' נטיפה והזלה ומושאל מענין הדבור וכן הטף אל דרום ( יחזקאל כ''א ) :
! {יז}
<b>בחבל .</b> קו המדה :

~ עמוס פרק-ח

! {א}
<b>כלוב .</b> הוא כעין סל וכן ככלוב מלא עוף ( ירמיה ה' ) : <b>קיץ .</b> ר''ל פירות המייבשים בימות הקיץ וכן והלחם והקיץ ( ש''ב ט''ז ) :
! {ג}
<b>והילילו .</b> מל' יללה : <b>שירות .</b> מל' שיר וזמר : <b>הפגר .</b> כן יקרא גוף המת כמו וכובד פגר ( נחום ג' ) : <b>הס .</b> ענין שתיקה כמו ואמר הס ( לעיל ו ) :
! {ד}
<b>השואפים .</b> ענין בליעה וההמשכה אל הפה כמו שאפני אנוש ( תהלים נ''ו ) : <b>ולשבית .</b> כמו ולהשבית וכן ללכת לגיד ( מ''ב ט' ) ומשפטו להגיד ולהשבית הוא ענין בטול ומניעה כמו שבת נוגש ( ישעיה י''ד ) :
! {ה}
<b>ונשבירה .</b> מוכר התבואה נקרא משביר כמו המשביר לכל עם הארץ ( בראשית מ''ב ) שבר כן תקרא התבואה וכן נאמר שבר רעבון בתיכם ( שם מ''ג ) וזהו לפי ששוברת הרעב על דרך ישברו פראים צמאם ( תהלים ק''ד ) : <b>והשבת .</b> כן תקרא שנת השמטה כמ''ש ובשנה השביעית שבת וגו' ( ויקרא כ''ה ) : <b>בר .</b> תבואה כמו לשבור בר ( בראשית מ''ב ) : <b>איפה .</b> שם מדה בת ג' סאים : <b>ולעות .</b> ענין עקום ודבר שאינו ראוי והוא מל' עון :
! {ו}
<b>נעלים .</b> מנעלי הרגל : <b>ומפל .</b> מל' הפלה :
! {ח}
<b>תרגז .</b> ענין תנועת הרעדה : <b>ואבל .</b> ענין השחתה כמו אבלה אדמה ( יואל א' ) : <b>כאור .</b> כמו כיאור הוא הנילוס ונפלה היו''ד : <b>ונגרשה .</b> מל' גרושין ותנודה . ונשקעה . ענין נפילת הדבר בעומק כמו אז אשקיע מימיהם ( יחזקאל לב ) :
! {ט}
<b>והבאתי .</b> ענין שקיעה כמו ובא השמש וטהר ( ויקרא כ''ב ) : <b>בצהרים .</b> הוא חצי היום :
! {י}
<b>קרחה .</b> ענין נפילה ותלישת השער כמו קרח הוא ( שם יג ) :
! {יב}
<b>ונעו .</b> מלשון תנועה : <b>ישוטטו .</b> ענין ההליכה אנה ואנה כמו שוטטו בחוצות ירושלים ( ירמיה ה' ) :
! {יג}
<b>תתעלפנה .</b> ענין החלשות והעדר ההרגשה כמו בניך עלפו ( ישעיה נ''א ) :
! {יד}
<b>באשמת .</b> מל' אשם וכן תקרא העכו''ם כן מאשמת עובדיה וכן תקרא פשע עש''ז כמ''ש והפשע שמה ( דניאל ח ) : <b>עוד .</b> ענין זמן רב ודוגמתו ויבך על צואריו עוד ( בראשית מ''ו ) :

~ עמוס פרק-ט

! {א}
<b>הכפתור .</b> הוא המשקוף וקראו כפתור לפי שעושין בו ציורי כפתורים העגולים כעין תפוחים וכן בכפתורים ילינו ( צפניה ב ) : <b>וירעשו .</b> ענין תנועה חזקה : <b>הספים .</b> הם המזוזות כמו וסף השער ( יחזקאל ט ) : <b>ובצעם .</b> חמדת עשרם כמו בצע כסף ( שופטים ה ) : <b>ואחריתם .</b> כן יקראו הבנים והבנות כמו ולא לאחריתו ( דניאל י''א ) : <b>ינוס .</b> ענין בריחה : <b>להם .</b> כמו מהם : <b>ימלט .</b> ענין השמטה : <b>פליט .</b> ענין שארית כמו פליט ושריד ( איכה ב' ) :
! {ב}
<b>יחתרו .</b> ענין חפירה בקרקע כמו חתר בחשך בתים ( איוב כ''ד ) : <b>בשאול .</b> בקבר עמוק :
! {ג}
<b>יחבאו .</b> ענין הסתרה כמו חבי כמעט רגע ( ישעיה לב ) : <b>אחפש .</b> ענין חקירה כמו ויחפש רוחי ( תהלים עז ) : <b>הנחש .</b> הוא הלויתן הדומה למראה נחש וכמ''ש על לויתן נחש בריח ( ישעיה כז ) : <b>ונשכם .</b> מל' נשיכה בשינים :
! {ה}
<b>ותמוג .</b> מל' המגה והמסה : <b>ואבלו .</b> ענין השחתה כמו אבלה אדמה ( יואל א ) : <b>ושקעה .</b> ענין נפילת הדבר בעומק כמו אשקיע מימיהם ( יחזקאל לב ) :
! {ו}
<b>מעלותיו .</b> מלשון עלייה הבנוי ממעל ואמר בלשון שאלה על הגלגלים : <b>ואגודתו .</b> ענין קבוצה אחת וכן יהיו לאגודה אחת ( ש''ב ב ) :
! {ז}
<b>כבני כושיים .</b> הם האנשים השחורים מבני כוש בן חם :
! {ח}
<b>אפס .</b> עניינו כמו רק וכן ואפס את הדבר ( במדבר כב ) :
! {ט}
<b>והניעותי .</b> מל' הנעה והנדה : <b>בכברה .</b> היא כעין נפה שמרקדין בו הקמח ודומה לו ועשית לו מכבר ( שמות כז ) : <b>צרור .</b> חתיכה גסה של עפר וכן אשר לא נמצא שם גם צרור ( ש''ב יז ) :
! {י}
<b>תגיש .</b> ענין הקרוב : <b>ותקדים .</b> ענין מהירות : <b>בעדינו .</b> ר''ל עלינו :
! {יא}
<b>וגדרתי .</b> ענין בנין : <b>פרציהן .</b> ענין שבר חומה : <b>והריסותיו .</b> מל' הריסה ונתיצה :
! {יב}
<b>למען .</b> ענינו כמו באופן כמו למען ינוח שורך ( שמות כג ) :
! {יג}
<b>ונגש .</b> ענין הקרוב והדבוק : <b>ודורך .</b> הדרך לדרוך על הענבים להוציא היין : <b>במושך .</b> מל' משיכה והולכה : <b>והטיפו .</b> ענין הזלה : <b>עסיס .</b> כן יקרא היין וכן על עסיס כי נכרת מפיכם ( יואל א ) : <b>תתמוגגנה .</b> מל' המגה והמסה :
! {יד}
<b>ושבתי .</b> ענין השקט ומרגוע כמו בשובה ונחת תושעון ( ישעיה ל ) : <b>שבות .</b> מלשון שבי הגולה : <b>נשמות .</b> מלשון שממון :
! {טו}
<b>ינתשו .</b> ענין עקירה כמו לנתוש ולנתוץ ( ירמיה א ) :

~ עמוס פרק-י

! {א}
<b>וציר .</b> ענין שליח כמו וציר אמונים ( משלי יג ) :
! {ב}
<b>בזוי .</b> מבוזה :
! {ג}
<b>השיאך .</b> ענין הסתה כמו הנחש השיאני ( בראשית ג ) : <b>שוכני .</b> היו''ד יתירה : <b>בחגוי .</b> ענין בקוע וחריץ כמו יונתי בחגוי הסלע ( ש''ה ב ) :
! {ד}
<b>קנך .</b> מל' קן והוא מדור העופות :
! {ה}
<b>שודדי .</b> ענין גזל ועושק : <b>נדמיתה .</b> ענין כריתה ואבדון כמו ער מואב נדמה ( ישעיה טו ) : <b>דים .</b> די הספוק
