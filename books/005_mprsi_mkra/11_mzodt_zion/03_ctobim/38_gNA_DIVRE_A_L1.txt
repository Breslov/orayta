<b>לחקור .</b> מל' חקירה וחפוש : <b>ולהפוך .</b> ת''י ולמבדק כי הבודק דבר מה מהפכה מעבר אל עבר : <b>ולרגל .</b> גם הוא ענין חקירה וחפוש :
! {ד}
<b>מדויהם .</b> מלבושיהם כמו וילבש שאול וכו' מדיו ( ש''א וז ) : <b>המפשעה .</b> כמו המפסעה בסמ''ך והוא מלשון פסיעה ור''ל הרגלים שפוסעים בהם והוא כינוי לעגבות שהיא ברגלים ממעל ושמה יכלה חצי המלבוש :
! {ה}
<b>יצמח .</b> יגדל :

~ דברי הימים א פרק-כ

! {א}
<b>ויצר .</b> מלשון מצור :
! {ב}
<b>מלכם .</b> העבודת כוכבים הקרויה מלכום או כפשוטו : <b>יקרה .</b> חשובה :
! {ג}
<b>וישר .</b> כלי המלא פגימות אשר יכרתו בו העצים קרוי משור כמו אם יתגדל המשור ( ישעיה י ) וע''ש שכורתים בו העצים ע''י גרירה קרוי ג''כ מגרה כמו מגררות במגרה ( מלכים א ז ) והיה די לומר וישר שפירש חתך במשור אבל לתוספות ביאור אמר במגרה וכן רתקו בזיקים ( נחום ג ) : <b>ובחריצי הברזל .</b> כלי ברזל עשוי חריצים כמו כי לא בחרוץ יודש קצח ( ישעיה כח ) : <b>ובמגרות .</b> גם הם כלים מלאים פגימות אבל קטנים הם מן המשור :
! {ד}
<b>הרפא .</b> הענקים כמו מיתר הרפאים ( דברים ג ) :
! {ה}
<b>כמנור אורגים .</b> הוא העץ העגול אשר האורג כורך בו חוטי השתי ובמשנה נקרא כובד ( שבת קג ) :
! {ו}
<b>להרפא .</b> להענקה :
! {ח}
<b>אל נולדו .</b> כמו אלה נולדו וכן כל התועבות האל ( ויקרא י''ח ) :

~ דברי הימים א פרק-כא

! {א}
<b>ויסת .</b> ענין פתוי :
! {ג}
<b>לאשמה .</b> היא מלשון שממה וכן עדרי הצאן נאשמו ( יואל א ) :
! {ה}
<b>מספר מפקד .</b> לתוספת ביאור כפל כמ''ש וכן אדמת עפר ( דניאל י''ב ) :
! {ח}
<b>נסכלתי .</b> מלשון סכלות ושטות :
! {יב}
<b>נספה .</b> ענין כליון כמו אסף אסיפם ( ירמיה ח ) :
! {טו}
<b>הרף .</b> מלשון רפיון :
! {טז}
<b>שלופה .</b> הוצאה מתערה :
! {כב}
<b>ותעצר .</b> ענין מניעה :
! {כג}
<b>והמוריגים .</b> כעין לוח עץ ובתחתיתו תחובים אבנים דקים ובו דשים התבואה וכן למורג חרוץ ( ישעיה מא ) :
! {כו}
<b>נדנה .</b> הוא נרתק החרב ודוגמתו בגו נדנה ( דניאל ו ) :
! {ל}
<b>נבעת .</b> מל' בעתה וחרדה :

~ דברי הימים א פרק-כב

! {ב}
<b>חוצבים .</b> כן יקראו כורתי האבנים ממקום גדולם : <b>גזית .</b> היא מל' גזיזה ור''ל כרותות ומסותתות ביושר :
! {ג}
<b>למסמרים .</b> ליתדות :
! {ט}
<b>ושקט .</b> ענין מנוחה :
! {יג}
<b>ואמץ .</b> גם הוא ענין חוזק : <b>תחת .</b> ענין פחד ושבר כמו אל תערוץ ואל תחת ( יהושע א ) :

~ דברי הימים א פרק-כג

! {ג}
<b>לגברים .</b> לאנשים :
! {ד}
<b>לנצח .</b> מלאכת השיר נקרא מלשון נצוח כי דרך בעלי השיר לנצח זה את זה בהרמת הקול ובהכרעת הנעימה : <b>ושוטרים .</b> הם הנוגשים וכופים לעשות דברי השופטים :
! {יא}
<b>לפקודה .</b> ענין מנוי וגזברות :
! {כט}
<b>המערכת .</b> מל' ערוכה וסדור : <b>ולרקיקי .</b> חררות דקות כמו רקיקי מצות ( ויקרא ב ) : <b>ולמחבת .</b> מנחה מעשה מחבת : <b>ולמרבכת .</b> ענין חליטה ברותחין כמו מרבכת תביאנה ( שם ו ) : <b>משורה .</b> היא מדת הלח כמו במדה במשקל ובמשורה ( שם יט ) :

~ דברי הימים א פרק-כד

! {א}
<b>.</b> :

~ דברי הימים א פרק-כה

! {א}
<b>.</b> :
! {ה}
<b>קרן .</b> ענין ממשלה וחתק והושאל מבע''ח בעלי הקרנות אשר יתחזקו בהם בהלחמם עם מי :
! {ז}
<b>מלומדי .</b> ענין ההרגל כמו וילמד לטרף טרף ( יחזקאל י''ט ) :
! {ח}
<b>לעומת .</b> נגד :
! {יא}
<b>ליצרי .</b> הוא צרי האמור למעלה וכן ימצא קצת שינוי השמות בשאר הנזכרים וכדרך המקרא :

~ דברי הימים א פרק-כו

! {א}
<b>.</b> :
! {לב}
<b>ויפקידם .</b> ענין מנוי וגזברות : <b>לכל דבר .</b> לכל ענין וכן על דבר פעור ( במדבר כה ) :

~ דברי הימים א פרק-כז

! {א}
<b>המחלקות .</b> מל' חלק :
! {ד}
<b>ומקלות .</b> כן שמו בוי''ו :
! {ו}
<b>השלישים .</b> שרים וממונים כמו ומבחר שלישיו ( שמות טו ) :
! {כג}
<b>נשא .</b> ענין מספר כמו כי תשא ( שם ל ) :
! {כד}
<b>החל .</b> התחיל : <b>ולא כלה .</b> לא השלים :
! {לב}
<b>וסופר .</b> חכם ובדרז''ל סופר מברך ובור יוצא ( חולין קח ) :

~ דברי הימים א פרק-כח

! {א}
<b>הסריסים .</b> ענין שררה :
! {ב}
<b>ולהדום .</b> הוא שרפרף הכסא הנתון תחת רגלי האדם כשהוא יושב כמו והארץ הדום רגלי ( ישעיה סו ) ואחז במשל מדרך האדם :
! {ט}
<b>יצר .</b> הוא ענין פעולה כמו מי יצר אל ( שם מד ) : <b>יזניחך .</b> ענין עזיבה כמו לו זנחתני ( תהלים מג ) :
! {יא}
<b>תבנית .</b> ענין תואר וצורה : <b>וגנזכיו .</b> מקום האוצרות והוא מלשון גניזה והטמנה :
! {יג}
<b>ולמחלקות .</b> ואת מחלקות והלמ''ד הוא במקום את כמו לכל כליו ( שמות כז ) ורבים כמוהו :
! {יז}
<b>והמזלגות .</b> שם כלי בעלת השנים ובו תוחבים הבשר להעלות מן היורה . והמזרקות . הספלים אשר יקבלו בהם הדם לזריקה : <b>והקשות .</b> סניפין כעין יתדות עמדו אצל השולחן מזה ומזה לצורך לחם הפנים כמ''ש בתורה : <b>ולכפורי .</b> גם המה כעין מזרקות וכן כפורי זהב משנים ( עזרא א ) וע''ש שהכהן הזורק מקנח ידו בשפתו קרוים כפורי ובדרז''ל ובעי לכפורי ידיה ( גיטין נו ) :
! {יח}
<b>מזוקק .</b> ענין צרוף כמו מזוקק שבעתים ( תהלים יב ) :
! {כ}
<b>תחת .</b> ענין פחד ושבר : <b>לכלות .</b> להשלים :

~ דברי הימים א פרק-כט

! {א}
<b>הבירה .</b> הארמון ועל המקדש יאמר :
! {ב}
<b>אבני פוך .</b> אבני יקר ומראיהם כעין הפוך והוא הכחול : <b>ורקמה .</b> יריעות מרוקמות במיני צבעונין : <b>יקרה .</b> חשובה שיש שם מין אבן חשוב לבנין :
! {ג}
<b>סגולה .</b> ענין אוצר כמו וסגולת מלכים ( קהלת ב ) :
! {ד}
<b>לטוח .</b> מל' טיחה ומשיחה כמו וטח את הבית ( ויקרא יד ) ואמר בדרך השאלה כי בטסים חפו הבית ולא ע''י טיחה ומשיחה כדרך שעושים האומנים ומרבית הזהב יוכיח . קירות . כותלי :
! {ה}
<b>חרשים .</b> אומנים : <b>למלאות .</b> ענין חנוך כמו ומלאת יד אהרן ( שמות כט ) :
! {ו}
<b>ואדרכונים .</b> שם מטבע ידוע אצלם וכן לאדרכונים אלף ( עזרא ח ) :
! {יא}
<b>והתפארת .</b> מל' פאר : <b>והנצח .</b> מל' נצחון : <b>וההוד .</b> ענין הדר ויופי :
! {יג}
<b>מודים .</b> מל' הודאה :
! {יד}
<b>נעצור כח .</b> נעכב אצלנו הכח כמו נעצרה נא אותך ( שופטים יג ) :
! {טו}
<b>ותושבים .</b> היושב זמן רב במקום אחד קרוי תושב מלשון ישב :
! {טז}
<b>ההמון .</b> ענין מרבית ההון כמו ומי אוהב בהמון ( קהלת ה ) :
! {יז}
<b>בוחן .</b> מל' בחינה ונסיון : <b>הנמצאו .</b> אשר נמצאו הה''א היא במקום אשר וכן ההקדיש שמואל ( לעיל כו ) :
! {יח}
<b>ליצר .</b> הוא ענין פעולה כמו מי יצר אל ( ישעיה מד ) וכן וכל יצר מחשבות לבו ( בראשית ז ) : <b>והכן .</b> מל' הכנה :
! {כ}
<b>ברכו .</b> שבחו והללו : <b>ויקדו .</b> כפפו הקדקוד :
! {כה}
<b>למעלה .</b> מל' עליה והרמה :
! {ל}
<b>והעתים .</b> מל' עת וזמן :
